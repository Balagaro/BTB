Beyond the Blocks
BTB is an Android based mobile game. This survivor game implements AI, which were meant to make the gameplay much more interesting and fun.
This game was developed and maintained by Hungarian students, for a workshop typed competition, called Dusza Árpád Programműhely.
We managed to learn a lot of new technologies, new IDEs and new programming languages - and of course - we had a lot of fun during the project.
We suggest everyone should give it a try, and please feed back your thought and ideas about this game.
Thank you, and have fun:)
